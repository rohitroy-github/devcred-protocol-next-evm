🧪 How to deploy (quick flow)
Deploy DevCredProfile
Deploy DevCredEscrow(profileAddress)
Transfer ownership:
await profile.transferOwnership(escrow.address)

🔄 MVP Flow Recap
Dev → mintProfile()
Client → createJob() (send ETH)
Client → assignDeveloper()
Dev → submitWork()
Client → approveWork()
Dev gets paid + reputation updated
